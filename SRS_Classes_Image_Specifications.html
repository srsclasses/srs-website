üí° Image Optimization Tips

Best Practices for Web Images:

  • Use WebP format where possible for 30-50% better compression than JPG/PNG
  • Compress all images using tools like:
    • TinyPNG (https://tinypng.com)
    • Squoosh (https://squoosh.app)
    • ImageOptim (Mac) or FileOptimizer (Windows)
  • Use descriptive filenames (e.g., 'mentor-rajesh-kumar.jpg' not 'IMG_1234.jpg')
  • Add alt text for all images to improve accessibility and SEO
  • Consider lazy loading for below-the-fold images to improve page load speed
  • Use responsive images with srcset attribute for different screen sizes
  • Maintain aspect ratios to prevent layout shifts during page load
  • Test on retina displays to ensure quality on high-DPI screens
